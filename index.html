Acerca de mí

Me llamo Renzo Sánchez y actualmente estoy cursando la carrera de Computación e Informática. Me interesa el desarrollo web, la inteligencia artificial y la seguridad informática. El desarrollo web es un proceso complejo que implica la creación de productos digitales en línea. También es un término que define la creación de sitios web para Internet o una intranet. Para conseguirlo se hace uso de tecnologías de software del lado del servidor y del cliente que involucran una combinación de procesos de base de datos con el uso de un navegador web a fin de realizar determinadas tareas o mostrar información.
